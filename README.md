# سينما الورشة

موقع غرف مشاهدة مباشرة مبني كنسخة ثابتة جاهزة للنشر على `GitHub Pages`.

## ماذا يفعل

- الصفحة الرئيسية تطلب الاسم ثم تنشئ غرفة سينما.
- منشئ الغرفة يحمّل الفيلم محلياً داخل المتصفح فقط.
- الفيلم لا يتم رفعه إلى Firebase Storage ولا يُحفظ داخل قاعدة البيانات.
- Firebase Realtime Database تُستخدم فقط للغرف والشات والحضور والإشارات.
- المشاهدون يستقبلون الفيلم كبث مباشر عبر `WebRTC`.
- الشات مباشر، ولكل عضو أفاتار عشوائي، مع قائمة حضور وتاج لمنشئ الغرفة.
- زر المايك موجود بجانب الإرسال، وبعد الموافقة يعمل المايك ويُسمع داخل الغرفة.

## قبل التشغيل

إذا ظهر لك `Permission denied` فهذا يعني أن قواعد `Realtime Database` ما زالت تمنع القراءة والكتابة.

افتح Firebase Console ثم:

1. ادخل إلى `Realtime Database`.
2. افتح تبويب `Rules`.
3. انسخ محتوى الملف `firebase-rtdb-rules.json`.
4. احفظ القواعد ثم أعد تحميل الموقع.

## النشر على GitHub Pages

هذا المشروع مجهز لمسار جميل مثل:

`https://cenima-alwarsha.github.io/live/1234`

وذلك لأن ملف `404.html` يعيد توجيه أي رابط غرفة مباشرة إلى التطبيق الرئيسي.

## ملاحظات تقنية مهمة

- تشغيل المايك و`WebRTC` يحتاج الموقع أن يكون على `https` أو `localhost`.
- المشروع الحالي يستخدم `STUN` فقط. إذا أردت أفضل ثبات عبر كل الشبكات مستقبلاً فالأفضل إضافة `TURN`.
- لأن البث هنا مباشر من جهاز منشئ الغرفة إلى المشاهدين، فعدد المشاهدين الكبير سيحتاج سرعة رفع إنترنت قوية عند المنشئ.
